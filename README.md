# Community Hero - Hyperlocal Problem Solver

AI-powered platform for citizens to report and track community issues.

## Features
- Image-based issue reporting
- AI categorization using Gemini
- Geo-location mapping
- Community verification
- Real-time tracking

## Technologies Used
- Google AI Studio
- Gemini Vision API
- Google Maps API
- Firebase

## How to Run
Open index.html in browser

## Setup

1. Create a Google Gemini API key from Google AI Studio.

2. Open `app.js`.

3. Find this line:

```javascript
const GEMINI_KEY = "YOUR_GEMINI_API_KEY";
```

4. Replace `"YOUR_GEMINI_API_KEY"` with your own Gemini API key.

5. Save the file and refresh the website.
